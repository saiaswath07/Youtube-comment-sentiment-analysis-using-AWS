# YouTube Comment Sentiment Analysis Using AWS
Welcome to the YouTube Comment Sentiment Analysis project! This repository showcases a project that uses AWS services to perform sentiment analysis on YouTube comments.

## Overview
This project leverages the power of AWS to analyze the sentiment of comments on YouTube videos. It uses various AWS services like AWS Lambda, AWS Comprehend, and AWS S3 to fetch, process, and analyze the data.

## Features
Sentiment Analysis: Detects positive, negative sentiments in YouTube comments.
visualization: displays percentage dashboard of classes
